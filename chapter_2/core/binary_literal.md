### *2進数リテラル*
---
#### 概要
C++11までの数値表現リテラルは、8進数/10進数/16進数のリテラルが定義されていたが、2進数リテラルは定義されていなかった。
C++14では、遂に2進数リテラルが追加された。

2進数リテラルを使用する場合は、以下のどちらかの構文を使用する。
```
0b<binary digits sequence>
0B<binary digits sequence>
```

他の数値リテラル同様に、任意の整数型で表現できなければならない。

```c++
int main() {
    int a = 0b0101010;
    int b = 0b1010101;
    return 0;
}
```
